# .NET Core with Dapper
A starter .NET Core project with Dapper & MySQL and some preset features:

- Controller with authorization
- NLog
- AutoMapper
- Swagger
- User secrets
- AppSettings info in object class
- MemCached
- Sentry
- AWS S3
- ClientCaching
- Localization
- API versioning
- AWS DynamoDB
